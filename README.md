# SocketDemo

## Demo android project of using `SOCKET.IO`

## DESCRIPTION:
This project can communicate with `https://socket-io-chat.now.sh/` using `SOCKET.IO`

The events implemented are:
- CONNECT
- DISCONNECT
- CONNECTION ERROR
- ADD USER
- NEW MESSAGE
- JOIN USER
- LEAVE USER

## Installation
Just clone the repo and run on Android Studio!
Happy coding!
